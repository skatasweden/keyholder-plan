Which views are essential for the accounting program?

Select all that should be in the initial version. We can add more later.

A

SIE4 Import + Overview

Upload SIE4 file, see import progress, company info summary, financial years, parse errors. The landing page after import.

B

Chart of Accounts (Kontoplan)

Browse all accounts with BAS grouping (1xxx Assets, 2xxx Liabilities, 3xxx Revenue, etc.), account types, SRU codes. Searchable and filterable.

C

General Ledger (Huvudbok)

Per-account transaction list with running balance. Filter by date range, account, dimension. The core accounting view.

D

Voucher Browser (Verifikationer)

List all vouchers by series (A, B, C...), click to see debit/credit rows, dimensions, attachments. Search by description, amount, date.

E

Balance Sheet (Balansrapport)

Standard Swedish balance sheet grouped by BAS account classes. Shows IB, UB, and change. Matches Fortnox layout.

F

Income Statement (Resultatrapport)

Revenue - Costs = Result, grouped by BAS classes. Period comparison (month, quarter, YTD). Matches Fortnox layout.

G

Validation Dashboard

Run the 10 validator checks visually. Side-by-side comparison with Fortnox source data. Red/green status for each check. The "trust but verify" view.

H

Dimensions & Cost Centers

Browse dimensions (cost centers, projects, employees), objects per dimension, and per-object balances (OIB/OUB). Filter vouchers by dimension.
