Key design decisions

  • Sidebar navigation — always visible, grouped: Data (Import, Kontoplan, Huvudbok, Verifikationer) + Rapporter (Balans, Resultat) + Verktyg (Validering, Dimensioner)
  • Company switcher at sidebar top — dropdown with company name + org number
  • Page header — Fraunces heading + company/year context line
  • Stat cards — overview page shows key metrics at a glance
  • Warm beige background (#f0e9df), white cards, 1.5px borders, 18-20px radius
